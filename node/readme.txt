接口准备安装一下组件，以下组件我均已经准备批处理文件
1、首先运行npm install 双击install.bat，安装接口依赖组件

2、npm install express    双击express.bat 等云星网城

//下面为json解析工具的安装
3、npm install body-parser 双击body-parser.bat

4、npm install multer 双击multer.bat

//mysql数据库组件的安装
5、npm install mysql 双击mysql.bat

全部运行完成之后

6、运行 runservice.bat

就可以运行接口
